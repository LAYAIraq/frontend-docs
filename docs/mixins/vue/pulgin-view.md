PluginView
===============
This component uses mixin to populate plugin data on create, and inject properties for plugin edit component

## Structure

Component Functionality
---------
### Props
- `onFinish`
  - `type`: `[Function, Array]`
  - `default`: return void 
- `viewData`
  - `type`: `Object`
  - `default`: return null 
- `editPreview`
  - `type`: `Boolean`
  - `default`: set to false
